🚀 Deployment Guide

Complete guide to deploy Bajaj Pest Control website to production with SSL, custom domain, and monitoring.

☁️ Option 1: Vercel (Recommended) Free Recommended

Vercel is the fastest way to deploy. Auto SSL, global CDN, and instant deploys from Git.

1
Install Vercel CLI
npm install -g vercel
2
Create vercel.json (already done)
// This configures routing for the Express server
{
  "version": 2,
  "builds": [{"src": "server.js", "use": "@vercel/node"}],
  "routes": [{"src": "/(.*)", "dest": "/server.js"}]
}
3
Deploy to Vercel
cd C:\Users\Dell\Desktop\bajaj-final
vercel --prod
# Follow prompts: create account, link project
# Your site will be live at: https://bajaj-final.vercel.app
4
Add Environment Variables in Vercel Dashboard

Go to: vercel.com → Project → Settings → Environment Variables

# Add each variable from your .env file:
ADMIN_PASSWORD = bajaj@admin2024
SMTP_HOST      = smtp.gmail.com
SMTP_USER      = your-email@gmail.com
SMTP_PASS      = your-app-password
NOTIFY_EMAIL   = bajajpeastcontol1@gmail.com
5
Add Custom Domain
vercel domains add bajajpestcontrol.in
# Then point your domain DNS:
# A Record → 76.76.21.21
# CNAME www → cname.vercel-dns.com

🖥️ Option 3: VPS Server (DigitalOcean/AWS) Paid
1
Upload files to server
scp -r C:\Users\Dell\Desktop\bajaj-final root@YOUR_SERVER_IP:/var/www/bajaj/
2
Install dependencies & run with PM2
ssh root@YOUR_SERVER_IP
cd /var/www/bajaj && npm install
npm install -g pm2
pm2 start server.js --name bajaj-ppc
pm2 startup && pm2 save
# App will auto-restart on server reboot
3
Configure Nginx reverse proxy + SSL
# Install Nginx + Certbot
sudo apt install nginx certbot python3-certbot-nginx

# Create nginx config at /etc/nginx/sites-available/bajaj
server {
    server_name bajajpestcontrol.in www.bajajpestcontrol.in;
    location / {
        proxy_pass http://localhost:8083;
        proxy_http_version 1.1;
        proxy_set_header Host $host;
    }
}

# Enable SSL (free Let's Encrypt)
sudo certbot --nginx -d bajajpestcontrol.in -d www.bajajpestcontrol.in

  • ☐ Submit test form — verify lead appears in Admin Dashboard

  • ☐ Test WhatsApp FAB button

  • ☐ Replace GA4_MEASUREMENT_ID in index.html and analytics.js
