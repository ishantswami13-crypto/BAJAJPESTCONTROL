📖 Usage Guidelines
✅ Do — Green (#1B7D4B)
  • Use for primary CTA buttons ("Book Inspection")
  • Trust badges and safety certifications
  • Service icons and category markers
  • Active navigation links and states
  • Progress indicators and success states
  • Highlight pricing and value propositions
❌ Don't — Green (#1B7D4B)
  • Don't use for error or warning messages
  • Don't place green on green backgrounds
  • Don't use for destructive actions (delete)
  • Don't overuse — loses emphasis
  • Don't use light green for body text
  • Don't use white text on light green
✅ Do — Navy (#1A2332)
  • All headlines and subheadings
  • Navigation bar background
  • Footer background
  • Authority text and credentials
  • Table headers and data labels
  • Card borders and dividers
❌ Don't — Navy (#1A2332)
  • Don't use navy text on dark backgrounds
  • Don't use navy on navy (obvious but common)
  • Don't use for success/positive messages
  • Don't use for interactive elements alone
  • Don't use very small text in navy on gray
  • Don't use navy for error states
✅ Do — Gold (#D4AF37)
  • Certification and trust badges
  • Star ratings and review highlights
  • Premium service labels
  • Award and achievement markers
  • Accent borders on featured cards
  • Highlight numbers and statistics
❌ Don't — Gold (#D4AF37)
  • Never use white text on gold (fails WCAG)
  • Don't use gold for CTA buttons
  • Don't use gold on yellow/warm backgrounds
  • Don't overuse — kills the premium feel
  • Don't use for error or warning states
  • Don't use gold for small body text

:root { --color-primary: #1B7D4B; --color-primary-hover: #155A3D; --color-primary-tint: #F0FAF5; --color-navy: #1A2332; --color-gold: #D4AF37; --color-white: #FFFFFF; --color-success: #4CAF50; --color-warning: #FF9800; --color-error: #F44336; }

/* CTA Button */ .btn-primary { background: var(--color-primary); color: var(--color-white); } .btn-primary:hover { background: var(--color-primary-hover); } /* Heading */ h1, h2 { color: var(--color-navy); } /* Gold Badge */ .badge { background: var(--color-gold); color: var(--color-navy); }
