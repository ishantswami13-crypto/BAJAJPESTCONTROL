🆘 Troubleshooting Common Issues
🔴
Server not startingCheck: node server.js — look for error messages. Usually missing .env or npm install needed.
📧
Emails not sendingVerify SMTP_USER and SMTP_PASS in .env. For Gmail: use App Password, not login password. Enable 2FA first.
💾
Leads not savingCheck data/ folder exists and is writable. View server logs for errors.
🔐
Can't login to adminCheck ADMIN_PASSWORD in .env. Default: bajaj@admin2024. Clear browser localStorage and retry.
🌐
Website slowRun Lighthouse audit in Chrome DevTools (F12 → Lighthouse tab). Check images are not too large.
